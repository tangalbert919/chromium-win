# chromium-win
-------------------------------

This repository is for stable Chromium builds for Windows. That's it.

## Build configuration
-------------------------------

* Visual Studio 2017
* No Widevine support
* PGO phase 1
* No Google API keys
* Official build configuration
* All proprietary codecs enabled
* Field trials disabled
* No RTC build examples
* Thin LTO enabled
* OpenH264 enabled
* WebRTC H.264 disabled
